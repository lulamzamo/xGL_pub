# isiXhosa Lexicalised Probabilistic Graphical Lemmatiser

This is a new implementation of the isiXhosa Graphical Lemmatiser (XGL). The original implementation was hosted on a platform that was discontinued. 

The work was rushed. Any identified bugs can be sent to the author, Lulamile Mzamo, at lula[underscore]mzamo[at]yahoo.co.uk
